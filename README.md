# Resume Site

This is a static resume website built with plain HTML, CSS, and JavaScript.

## Local run

```bash
npm start
```

The local server runs on `http://127.0.0.1:4173` when `PORT=4173` is set, or `8085` by default.

## Edit resume content

Update the `resumeData` object in `script.js` with your real:

- summary
- skills
- education
- experience
- projects

## Free hosting options

This project can be deployed as-is to any static host:

- Vercel
- Netlify
- Cloudflare Pages
- GitHub Pages

No build step is required. The publish directory is the project root.
