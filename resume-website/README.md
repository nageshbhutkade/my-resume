# Nagesh Bhutkade — Portfolio / Resume Website

A modern, responsive portfolio and resume website built with **HTML5**, **CSS3**, and **Vanilla JavaScript**.

## Features

- 🌓 **Dark / Light Theme** — with persistence via `localStorage`
- 📱 **Fully Responsive** — works on mobile, tablet, and desktop
- ⚡ **Performance Optimized** — fast loading with optimized animations
- ♿ **Accessible** — semantic HTML, ARIA labels, keyboard-friendly
- 🔍 **SEO Friendly** — Open Graph tags, semantic structure, meta tags
- 🎨 **Modern Design** — gradient accents, smooth transitions, card-based layout

## Sections

1. **Home** — Hero section with avatar, social links, and CTA buttons
2. **About** — Bio, stats, and personal details
3. **Skills** — Skill cards with tags and animated progress bars
4. **Experience** — Timeline layout with 4 work experiences
5. **Projects** — Filterable project cards with categories
6. **Education** — Academic background
7. **Contact** — Contact info cards + functional contact form (via Formspree)

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Grid, Flexbox, animations, responsive design
- **Vanilla JS** — Theme toggle, mobile nav, scroll animations, project filter
- **Font Awesome 6** — Icons
- **Google Fonts (Inter & JetBrains Mono)** — Typography
- **Formspree** — Form backend

## Getting Started

1. Clone or download the repository
2. Open `index.html` in your browser
3. No build tools or dependencies required!

To deploy:
- Upload all files to any static hosting (GitHub Pages, Netlify, Vercel, etc.)
- Or serve with any HTTP server

## Customization

- **Theme colors** — Edit the `--hue` CSS variable (currently `220` for blue)
- **Content** — Update text in `index.html`
- **Contact form** — Replace `nagedesh` in the Formspree action URL
- **Resume PDF** — Update the download link in the hero section
